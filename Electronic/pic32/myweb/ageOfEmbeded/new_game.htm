Games Selection

Which game do you want to play ? You ought to select at least one (or else we will choose for you...).

Play Speed
is a game where 12 boxes will appear on you screen. The goal will be to touch all of them in a minimum amount of time.

Play Pattern
is a game where a pattern will appear on you screen. The goal will be to redo the pattern without any error in a minimum amount of time

Play Curve Fever
is a game where each player control a snake. The first one that touch him self, a boarder or the enemy lost the fight
